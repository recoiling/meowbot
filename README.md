

## Setup

### Requirements

-   Python 3.10+
-   A Discord bot token
-   `pip install -r requirements.txt`

### Environment Variables

Create a `.env` file in the dir and add:

    TOKEN=
    PREFIX=
    COLOR=
    OWNER=


------------------------------------------------------------------------

## Features

### General

-   **help** -- Interactive help menu that groups commands by category.
-   **ping** -- Shows bot latency.
-   **uptime** -- Shows how long the bot has been running.

### Fun

-   **meow** -- Sends a random meow line and increases your meow count.
-   **meowlb** -- Shows the meow leaderboard.
-   **meowstats** -- Shows your personal meow count.
-   **fact** -- Fetches a random fact.
-   **slots** -- Slot machine game.
-   **reverse** -- Reverses your text.
-   **ship** -- Generates a compatibility percentage.
-   **cat** -- Sends a random cat image.

### Utilities

-   **serverinfo** -- Shows info about the current server.
-   **servericon / si** -- Displays the server icon.
-   **userinfo / ui** -- Shows info about a user.
-   **time / tz** -- Shows the time in a given timezone.
-   **define / def** -- Defines a word.
-   **urban / ud** -- Urban Dictionary search.

### Moderation

-   **purge** -- Clears messages. Optional user filter.
-   **clearbot / cb** -- Clears messages from the bot.
-   **lock** -- Locks a channel.
-   **unlock** -- Unlocks a channel.
-   **slowmode** -- Sets slowmode for the channel.
-   **temprole** -- Gives a temporary role for a set duration.

### Owner Commands

-   **addmeow** -- Add a new entry to the meow list.
-   **servers / guilds** -- List all servers the bot is in.

### Misc

-   **refreshpanel** -- Refreshes server metadata.
-   **say** -- Make the bot send a message.
-   **serverage / sage** -- Shows the age of the server.
